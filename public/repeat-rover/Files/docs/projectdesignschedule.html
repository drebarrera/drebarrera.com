EPIC I: IDEATION AND VALIDATION

The goals for the first Epic are to receive project proposal validation, ideate solutions to project deficits, and analyze the functionality of the proposed embedded system.

In this Epic, the team conceived of the Repeat Rover - a mobile solution to autonomous resource delivery for repeated process environments. The proposal was approved and a functional specification document was produced to outline development constraints.

EPIC II: DESIGN AND PROTOTYPING

The goals for the second Epic are to research strategies, components, and solutions to the embedded hardware problems faced by the Repeat Rover project.

In this Epic, the team designed the schematics and PCB layout of the Repeat Rover system. Hardware components were prototyped using a breakout version of the nRF52832 microcontroller to be used in the final design. This Epic concluded with the finalization of a PCB design and MCU-tested hardware components for construction and programming in the third Epic.
